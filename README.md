# j0rpiPlayer

Simple music player using NAudio, TagLibSharp and BASS.NET

## Features
- Simple media playback using NAudio
- ID3Tag support using TagLib.Sharp
- Tracker module support using BASS.NET
  - .mod
  - .xm
  - .it
  - .s3m
  - .mtm
  - .umx
  - .mo3
- Discord Rich Presence
